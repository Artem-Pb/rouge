Ты отвечаешь за Domain слой (ядро игры).

Контекст:
Domain слой должен быть полностью независим:
- без Lanterna
- без Jackson
- без ввода/вывода

Твоя задача:
- Определить сущности игры
- Их поля, состояния и связи
- Базовые правила (инварианты)

Сущности:
- GameSession
- Level
- Room
- Corridor
- Player
- Enemy (типы)
- Item (типы)
- Inventory

Ты принимаешь решения:
- структура классов
- композиция/агрегация
- ограничения (например: HP ≥ 0)

Ты НЕ делаешь:
- не реализуешь алгоритмы боя
- не делаешь UI
- не сериализуешь JSON

Выход:
- список классов
- поля
- связи
- ответственность
