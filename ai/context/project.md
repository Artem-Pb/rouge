# Project Context — Roguelike Game (Java Bootcamp)

## 📌 Общая идея

Проект представляет собой консольную roguelike игру, вдохновленную классической игрой Rogue (1980).

Игрок управляет персонажем, исследует подземелья, сражается с противниками, собирает предметы и пытается пройти 21 уровень.

Каждое прохождение уникально благодаря процедурной генерации уровней.

---

## ⚙️ Технологии

- Язык: Java 21
- UI: Lanterna (terminal-based UI)
- Data: Jackson (JSON сериализация)
- Тестирование: JUnit 5

---

## 🏗️ Архитектура

Приложение должно использовать многослойную архитектуру:

### 1. Domain Layer
- Содержит сущности игры
- Не зависит от других слоев
- Не использует внешние библиотеки

### 2. Gameplay Layer
- Содержит игровую логику
- Управляет поведением сущностей
- Использует domain, но не зависит от UI и data

### 3. Presentation Layer
- Отвечает за отображение игры
- Использует библиотеку Lanterna
- Обрабатывает ввод пользователя
- НЕ содержит бизнес-логики

### 4. Data Layer
- Отвечает за сохранение и загрузку данных
- Использует Jackson
- Работает с JSON

---

## 🎮 Основная механика игры

### Игровой процесс
- Игра состоит из 21 уровня
- Каждый уровень — подземелье из 9 комнат (3x3)
- Комнаты соединены коридорами
- Все комнаты должны быть достижимы (связный граф)

### Цель
- Найти выход на следующий уровень
- Пройти все 21 уровень
